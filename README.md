# How to run
1. Load environment variables
```
source setenv.sh # linux, mac

or 

setenv.bat # windows
```

2. install dependencies
```
pip install -r requirements.txt
```

3. run server
```
python manage.py runserver
```

4. open [localhost:8000](http://localhost:8000)
